# Книга рецептов (FastAPI)

Веб-приложение для управления продуктами и блюдами с отдельной серверной частью на FastAPI и встроенной SQLite-базой.

## Что реализовано

- CRUD продуктов.
- CRUD блюд с составом (многие-ко-многим через `dish_ingredients` с количеством в граммах).
- Фильтрация/поиск/сортировка продуктов.
- Фильтрация/поиск блюд.
- Автоматический расчет КБЖУ блюда по составу, с возможностью ручной корректировки.
- Макросы категории в названии блюда (`!десерт`, `!первое`, `!второе`, `!напиток`, `!салат`, `!суп`, `!перекус`) с удалением макроса из названия.
- Ограничение выставления флагов блюда по составу.
- Запрет удаления продукта, если он входит в состав хотя бы одного блюда (с выдачей списка блюд).
- API и базовые тесты (`pytest` + `TestClient`).

## Запуск

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
uvicorn app.main:app --reload
```

Откройте `http://127.0.0.1:8000`.

## Тесты

```bash
pytest -q
```

## Unit-тесты расчёта КБЖУ (по ТЗ)

Unit-тесты для автоматического расчёта калорийности блюда находятся в `tests/test_unit_dish_macros.py`.

Запуск:

```bash
source .venv/bin/activate
pytest -q
```
